## Endpoints

1. **AUTH:**
   - /auth
   -       /Signin    (POST)
   -       /Signup    (POST)


2. **IMAGES:**
   - /images              (GET)/(POST)
   -       /:imageid      (GET)/(DELETE)
   -             /react   (GET)/(POST)

3. **USERS:**
   - /users                (GET)
   -       /:userid        (GET)
   -             /images   (GET)
